# API Reference

The Badget API provides programmatic access to all link management features. This REST API uses JSON for request and response data.

## Base URL

All API requests should be made to:

```
https://api.badget.app
```

## Authentication

Authenticate your API requests using a Bearer token in the Authorization header:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.badget.app/links
```

### Getting Your API Key

1. Log in to your Badget dashboard
2. Navigate to Settings â†’ API Keys
3. Click "Generate New Key"
4. Copy and securely store your key

## Rate Limiting

API requests are limited to:

- **Free Plan**: 1,000 requests per hour
- **Pro Plan**: 10,000 requests per hour
- **Enterprise**: Custom limits

Rate limit headers are included in all responses:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200
```

## Links

### Create a Link

Create a new shortened link.

```http
POST /links
```

**Parameters:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `url` | string | Yes | The destination URL |
| `alias` | string | No | Custom alias for the short link |
| `title` | string | No | Link title for tracking |
| `tags` | array | No | Array of tag strings |
| `expiresAt` | string | No | ISO date when link expires |

**Example Request:**

```bash
curl -X POST https://api.badget.app/links \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com",
    "alias": "my-link",
    "title": "Example Website",
    "tags": ["example", "test"]
  }'
```

**Example Response:**

```json
{
  "id": "clr123abc",
  "shortUrl": "https://bdg.et/my-link",
  "url": "https://example.com",
  "alias": "my-link",
  "title": "Example Website",
  "tags": ["example", "test"],
  "clicks": 0,
  "createdAt": "2024-01-15T10:30:00Z",
  "expiresAt": null
}
```

### Get Link Details

Retrieve details for a specific link.

```http
GET /links/{id}
```

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.badget.app/links/clr123abc
```

### Update a Link

Update an existing link's properties.

```http
PUT /links/{id}
```

**Example Request:**

```bash
curl -X PUT https://api.badget.app/links/clr123abc \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Updated Title",
    "tags": ["updated", "example"]
  }'
```

### Delete a Link

Delete a link permanently.

```http
DELETE /links/{id}
```

**Example Request:**

```bash
curl -X DELETE https://api.badget.app/links/clr123abc \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### List Links

Get a paginated list of your links.

```http
GET /links
```

**Query Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `limit` | number | Number of links to return (max 100) |
| `offset` | number | Number of links to skip |
| `tag` | string | Filter by tag |
| `search` | string | Search in title and URL |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.badget.app/links?limit=10&tag=example"
```

## Analytics

### Get Link Analytics

Retrieve analytics data for a specific link.

```http
GET /analytics/links/{id}
```

**Query Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `period` | string | Time period: `1h`, `24h`, `7d`, `30d`, `90d` |
| `interval` | string | Data interval: `hour`, `day`, `week` |
| `country` | string | Filter by country code |
| `referrer` | string | Filter by referrer domain |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.badget.app/analytics/links/clr123abc?period=7d&interval=day"
```

**Example Response:**

```json
{
  "id": "clr123abc",
  "totalClicks": 1250,
  "uniqueClicks": 890,
  "clicksByDate": [
    {
      "date": "2024-01-15",
      "clicks": 180,
      "uniqueClicks": 150
    }
  ],
  "topCountries": [
    {
      "country": "US",
      "clicks": 450
    },
    {
      "country": "GB",
      "clicks": 200
    }
  ],
  "topReferrers": [
    {
      "referrer": "twitter.com",
      "clicks": 300
    },
    {
      "referrer": "linkedin.com",
      "clicks": 150

  ]
}
```

## Webhooks

Set up webhooks to receive real-time notifications about link events.

### Webhook Events

- `link.clicked` - When a link is clicked
- `link.created` - When a new link is created
- `link.updated` - When a link is updated
- `link.deleted` - When a link is deleted

### Example Webhook Payload

```json
{
  "event": "link.clicked",
  "data": {
    "linkId": "clr123abc",
    "shortUrl": "https://bdg.et/my-link",
    "clickedAt": "2024-01-15T14:30:00Z",
    "country": "US",
    "referrer": "twitter.com",
    "userAgent": "Mozilla/5.0..."
  }
}
```

## Error Handling

The API uses conventional HTTP response codes:

- `200` - Success
- `400` - Bad Request
- `401` - Unauthorized
- `403` - Forbidden
- `404` - Not Found
- `429` - Rate Limited
- `500` - Internal Server Error

**Error Response Format:**

```json
{
  "error": {
    "code": "INVALID_URL",
    "message": "The provided URL is not valid",
    "details": "URL must include http:// or https://"
  }
}
```

## SDK and Libraries

Official libraries are available for popular programming languages:

- **JavaScript/Node.js**: `npm install @badget/js`
- **Python**: `pip install badget`
- **PHP**: `composer require badget/php`
- **Go**: `go get github.com/badget/go`

### JavaScript Example

```javascript
import { Badget } from '@badget/js';

const badget = new Badget('YOUR_API_KEY');

// Create a link
const link = await badget.links.create({
  url: 'https://example.com',
  alias: 'my-link'
});

console.log(link.shortUrl); // https://bdg.et/my-link
```
